# ☁️ Cloud Computing Notes (Beginner → Intermediate)

## 📌 What is Cloud Computing?
Cloud computing is the **on-demand delivery of IT resources over the internet** with **pay-as-you-go pricing**.

Instead of maintaining physical data centers, you can use cloud providers like:
- AWS (Amazon Web Services)
- Microsoft Azure
- Google Cloud Platform (GCP)

---

## 🏗️ Traditional Data Center (On-Premises)

To host applications, you need:
- Servers
- Networking Devices
- Rack Space
- Power & Cooling
- Maintenance Team

👉 This is called **CAPEX (Capital Expenditure)**

---

## ☁️ Benefits of Cloud Computing

### 💰 1. Cost Efficiency
- Traditional: CAPEX
- Cloud: OPEX (Operational Expense)

Example:
- Buying a car → CAPEX  
- Fuel & maintenance → OPEX  

---

### ⚡ 2. Elasticity
- Scale resources up/down based on demand
- No need to pre-plan capacity

---

### 🚀 3. Speed & Agility
- Launch servers in minutes
- No hardware setup required

---

### 📉 4. No Capacity Guessing
- Automatically adjust based on traffic

---

### 🌍 5. Global Reach
- Deploy applications worldwide
- Low latency for users

---

## 🌐 Types of Cloud

### 🔒 Private Cloud
- Used by one organization
- High security
- Example: Banking systems

---

### 🌍 Public Cloud
- Owned by third-party providers (AWS, Azure, GCP)
- Shared infrastructure

**Limitation:**
- Less control
- Region failure can impact services

---

### 🔀 Hybrid Cloud
- Combination of Private + Public Cloud

**Example:**
- Normal load → Private Cloud  
- High traffic → Public Cloud  

---

## 📦 Cloud Service Models

---

### 1️⃣ IaaS (Infrastructure as a Service)

**You Manage:**
- OS
- Middleware
- Runtime
- Applications
- Data

**Cloud Provider Manages:**
- Servers
- Storage
- Networking
- Virtualization

📌 Example: AWS EC2

---

### 2️⃣ PaaS (Platform as a Service)

**You Manage:**
- Applications
- Data

**Cloud Provider Manages:**
- Runtime
- Middleware
- OS
- Infrastructure

📌 Example: AWS Elastic Beanstalk

---

### 3️⃣ SaaS (Software as a Service)

**You Manage:**
- Nothing (just use the software)

**Cloud Provider Manages:**
- Everything

📌 Examples:
- Microsoft Word
- Microsoft Excel
- Google Docs

## 🧠 Key Terms

- **Elasticity** → Auto scaling resources
- **High Availability** → No downtime
- **Disaster Recovery** → Backup region
- **Pay-as-you-go** → Cost optimization
